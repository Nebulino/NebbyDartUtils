## Get NebbyDartUtils

Add **NebbyDartUtils** dependency on `pubspec.yaml`:

From GitHub:
```yaml
dependencies:
  NebbyDartUtils:
    git: https://github.com/Nebulino/NebbyDartUtils.git
      ref: master
```
